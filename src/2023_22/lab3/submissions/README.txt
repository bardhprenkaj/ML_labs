Please upload your solution to the exercise by opening a pull request to this repo.
The name of the file can be any nickname that you want to associate with yourself.
For example a file can be named ver2b.ipnyb / ver2b.py

Notice that you need to put your solution under the lab3/submissions folder.

Please read the description of the exercise carefully. Some things in PyTorch
(e.g., hyperparameter optimization and data loading) might change w.r.t. scikit-learn.

Use Exercise_Implement_a_Multiclass_Classificaiton_Neural_Network.ipnyb template to begin
implementing your solution.

Have fun!
